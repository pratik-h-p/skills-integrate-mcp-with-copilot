# Add Activity CRUD and admin dashboard

Description
-----------
Implement API endpoints and a simple admin UI to create, edit, and delete activities (name, description, schedule, capacity). Provide an admin-only dashboard to manage activities and view participant lists.

Acceptance criteria
-------------------
- Admin API endpoints for create/read/update/delete activities exist.
- A minimal web UI allows an admin to manage activities and view participants.
- Admin pages are not accessible to anonymous users (auth gating to be implemented separately).

Labels: frontend, backend, enhancement
